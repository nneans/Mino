# Mino

개인 금융 관리의 혁신, **Mino** 통합 저장소입니다.

## 📂 저장소 구조

- **`/App`**: Mino 데스크탑 애플리케이션 (Node.js/Electron)
- **`/Web`**: 공식 홈페이지 및 정적 에셋
- **`/Releases`**: 최신 설치 파일 ([GitHub Releases](https://github.com/nneans/Mino/releases)에서 확인 가능)

## 🚀 빠른 시작

### 홈페이지 및 다운로드
- **공식 홈페이지**: [https://nneans.github.io/Mino](https://nneans.github.io/Mino)
- **데스크탑 앱 다운로드**: [최신 버전(.dmg) 다운로드](https://github.com/nneans/Mino/releases/latest)

## 🛠️ 주요 기능
- **AI 기반 지출 분석**: 자연어로 가계부 데이터를 조회하고 분석합니다.
- **다양한 동기화**: 결제 내역 및 예산을 간편하게 관리합니다.
- **시각화 리포트**: 차트와 지도를 통해 지출 패턴을 한눈에 파악합니다.
